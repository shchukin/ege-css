.gallery

Функционал галереи основан на скрытых радиокнопках. Одна радиокнопка отвечает за одну картинку. Когда она чекнута соответствующая картинка показывается, а все остальные скрыты. Сответствено для переключения слайдов нужно переключать радио-кнопки. Для этого делаем тумбнейлчики внизу тегом <label> и направляем их на соответствующую радиокнопку: свзяываем по id и for. За этим надо следить. Зато жаба-скрипт не нужен.

Сейчас есть ограничение на то, что в галерее может быть максимум 9 айтемов. Это связанно с тем, что для каждой радиокнопки нужно отдельно прописывать :checked состояние. Смотри конец файла styles/_gallery.css. Если вдруг надо будет больше, то можно вынести эти строчки в HTML и генерить програмно с бекенда.
